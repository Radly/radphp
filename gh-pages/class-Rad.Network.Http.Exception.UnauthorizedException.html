Class UnauthorizedException

Represents an HTTP 401 error.

Similar to 403 Forbidden, but specifically for use when authentication is required and has failed or has not yet been provided. The response must include a WWW-Authenticate header field containing a challenge applicable to the requested resource. 401 semantically means "unauthenticated" the user does not have the necessary credentials.
